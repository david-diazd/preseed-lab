# CMO — Marketing, Growth & Brand

Suite de agentes y skills para el rol de CMO/Head of Growth. Cubre marketing, growth, brand y adquisición de clientes.

## Cómo funciona

1. Pon tus documentos en `inputs/` (raíz del repo)
2. Invoca el agente con `@CMO/agents/<dominio>/<nombre>.md`
3. Los resultados se guardan en `outputs/`

## Flujo recomendado para lanzamiento

1. `@CMO/agents/brand/brand-positioning.md` — define el posicionamiento
2. `@CMO/agents/brand/messaging-framework.md` — crea el messaging
3. `@CMO/agents/marketing/landing-page.md` — optimiza la landing
4. `@CMO/agents/growth/growth-model.md` — diseña el motor de crecimiento
5. `@CMO/agents/marketing/launch-strategy.md` — planifica el lanzamiento
6. `@CMO/skills/campaign-launch.md` — ejecuta con el checklist
